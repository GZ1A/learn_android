# Android-NDK demo

NDK是一个Android官方提供的一个开发套件与Android　SDK相对应，NDK是原生开发套件，而SDK是JAVA开发套件。  
简而言之，我们做了一个解压软件，可以解压7z和zip的文件，这里主要是项目相关的概念介绍以及DEMO的展示和相关问题汇总。
